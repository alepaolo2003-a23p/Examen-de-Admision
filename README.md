# Sistema de Examen de Admisión - Backend API
Backend del Sistema de Examen de Admisión, desarrollado bajo una Arquitectura Modular. Este proyecto gestiona el registro de postulantes, banco de preguntas, generación de exámenes y calificación automática.

## 🚀 Tecnologías (Stack MERN - Fase Backend)
* **Entorno:** Node.js
* **Framework:** Express.js
* **Base de Datos:** MongoDB Atlas (Mongoose)
* **Gestor de paquetes:** Yarn

## ⚙️ Instalación y Configuración Local
1. Clonar el repositorio: `git clone https://github.com/alepaolo2003-a23p/Examen-de-Admision.git`
2. Instalar dependencias: `yarn install`
3. Configurar variables de entorno: Crea un archivo `.env` en la raíz y agrega `MONGO_URI=aqui_va_la_url` y `PORT=3000`.
4. Levantar servidor: `yarn dev`
